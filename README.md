# 📺 Multi-Language YouTube Subtitle Translator & Vocab Builder

A powerful Python tool that extracts subtitles from YouTube videos in various languages (English, Spanish, etc.), translates them into Korean, and automatically generates a vocabulary list for language learners.

## ✨ Key Features
- **Auto Language Detection:** Automatically lists all available subtitle tracks (manual & auto-generated) for any YouTube video.
- **Bi-lingual Output:** Saves both the original subtitle and the Korean translation line-by-line for easy comparison.
- **Smart Vocab Builder:** Automatically extracts and deduplicates words from the subtitles to create a custom study list.
- **Copyright Safe:** Designed to process data locally without storing copyrighted content in the repository.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
   cd YOUR_REPO_NAME
   ```

2. **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## 🚀 How to Use
1. Run the main script:
    ```bash
    python main.py
    ```

2. Enter the YouTube URL of the video you want to study (e.g., Bluey, TED Talks, News).

3. Select the language number from the detected subtitle list.

4. Check the generated .txt file for your translated script and vocabulary list!
